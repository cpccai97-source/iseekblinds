Step by Step

Measuring Inside the Recess

01

Measure the Width

Measure the width of the window recess at three points: top, middle and bottom. Record the narrowest measurement in millimetres. This ensures your blind fits without jamming.

Pro tip: Don't assume your window is square — most aren't. A 5mm difference is common.
02

Measure the Height

Measure the drop from the top of the recess to the windowsill at three points: left, centre and right. Record the longest measurement so your blind covers the entire window.

Pro tip: For roller blinds, add 10mm to the drop for proper overlap.
03

Check the Depth

Measure how deep the recess is from the window glass to the front edge. This determines which products can be fully recessed. Most roller blinds need at least 60mm depth.

04

Note Any Obstructions

Check for handles, locks, tiles or anything that might interfere with the blind. Measure the position and protrusion of each obstruction from the window frame edge.

05

Label Each Window

Give every window a unique label (e.g., "Master Bedroom – Left"). Draw a simple floor plan and mark each window. This prevents mix-ups during ordering and installation.

Pro tip: Take a photo of each window with the measurements written on a sticky note in frame.
By Product

Measuring Tips for Each Product Type

🔲

Roller Blinds

Measure recess width and drop. For face-fix (outside recess), add 50mm each side for light gap coverage. Specify chain side — left or right.

RECESS OR FACE-FIX
🪟

Venetian Blinds

Recess mount preferred — measure width at three points. Allow 10mm clearance. Depth must be ≥45mm for 25mm slats, ≥70mm for 50mm slats.

RECESS RECOMMENDED
🏠

Plantation Shutters

Measure recess width, height and depth. Check if walls are plumb with a spirit level. Note any tiles, skirting or architrave that may affect fitting.

PRECISION CRITICAL
✨

Curtains

Measure track/rod width (not window). For drop: measure from rod to floor, then subtract 10mm. For puddle effect, add 50–150mm instead.

ROD TO FLOOR
🐝

Honeycomb Blinds

Same as roller blind measurements. For top-down/bottom-up operation, ensure accurate drop measurement — these blinds are cut to exact size.

EXACT FIT REQUIRED
⚡

Motorised Blinds

Measure as per product type above, plus note the nearest power point location. Battery motors need 80mm+ recess depth for the motor housing.

POWER POINT LOCATION
Avoid These

Common Measuring Mistakes

✕

Measuring once

Single measurement misses variations in window frame

✓ Measure 3 points

Top, middle, bottom for width — left, centre, right for height

✕

Using a cloth tape

Fabric tape measures stretch and give inaccurate readings

✓ Steel tape measure

Rigid steel tape gives precise, reliable measurements

✕

Rounding measurements

Rounding to the nearest cm can mean 5mm+ error

✓ Record in millimetres

Always measure and record in mm for manufacturing precision

✕

Deducting for fit

Subtracting clearance yourself can lead to double-deduction

✓ Give exact recess size

We deduct the correct clearance for each product type
