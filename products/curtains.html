How To Measure

Curtain Measuring Guide

📏 Track Width

Measure the width of your window and add 150-200mm each side so curtains can stack off the glass when open.

📐 Drop Length

Measure from the track/rod position to the floor. For a puddled look, add 20-30mm. For a clean hem, subtract 10mm.
