Quick Summary

Skylight blinds are measured differently depending on whether you have a branded skylight (VELUX, Fakro, Roto) or a custom-built skylight.

  • Branded: Find the code plate on the frame → note the model code → done. No tape measure needed.
  • Custom: Measure width, height, and frame depth in mm. Note the roof pitch angle.
  • All skylights need side channels to stop the blind sagging away from the glass.
  • Honeycomb blinds are the best fit for most Australian skylights — lightweight, insulating, and angle-friendly.

First: Identify Your Skylight Type

Before grabbing the tape measure, work out what kind of skylight you're dealing with. This determines your entire measurement approach.

Branded Skylight

  • VELUX, Fakro, or Roto brand
  • Has a metal code plate on the frame
  • Standard sizes — just need the code
  • Purpose-made blinds clip straight in
  • Most common in Australian homes built after 1990

Custom-Built Skylight

  • Built by a glazier or carpenter on-site
  • No code plate — non-standard size
  • Requires precise manual measurement
  • May be fixed, openable, or tubular
  • Common in older homes and renovations
💡 Not sure? Open the skylight (if it opens) and look for a metal plate on the frame edge. If there's a code like "GGL MK04" or "FTP-V U3", you have a branded skylight. No plate = custom.

Finding the Code Plate (Branded Skylights)

This is the easiest part. Branded skylight manufacturers stamp a metal code plate on every unit during production. This plate contains everything you need to order the right blind.

Code plate locations on VELUX skylights. Open the skylight to access the plate.

How to Read the Code

The code plate typically shows:

  • Model prefix — e.g., GGL (centre-pivot, timber frame), GPU (top-hung, polyurethane), GGU (centre-pivot, polyurethane)
  • Size code — e.g., MK04, CK02, SK06. This two-part code defines the exact width and height.
  • Manufacture date — useful for identifying compatible blind systems
✅ Pro tip: Take a photo of the code plate with your phone. This avoids transcription errors and gives us all the detail we need if you request a quote.

Measuring Branded Skylights (VELUX, Fakro, Roto)

If you found a code plate, you're most of the way there. Branded skylight blinds are manufactured to match the exact frame dimensions of each size code.

1

Locate & Record the Code

Open the skylight and find the code plate (see diagram above). Write down or photograph:

  • The size code (e.g., MK04)
  • The model prefix (e.g., GGL, GPU)
  • The manufacture year (pre-2013 VELUX models use a different bracket system)
2

Check the Frame Depth

Even with a branded skylight, you need to verify the frame has enough depth for the blind type you want.

  • Honeycomb blinds: 25mm minimum depth
  • Roller blinds: 35mm minimum depth
  • Venetian blinds: 40mm minimum depth

Measure from the glass surface to the inside edge of the frame with a steel tape.

3

Note the Operation Type

How does your skylight open? This affects which blind systems are compatible:

  • Centre-pivot — blind must sit within the glass area and clear the pivot mechanism
  • Top-hung — more clearance, most blind types work
  • Fixed (non-opening) — any blind type works; simplest to fit
  • Electrically operated — consider motorised blinds to match (one remote for both)
✅ That's it for branded skylights. Send us the code plate photo and we'll handle the rest. No room for measurement error because the blind is factory-matched to your skylight model.

Measuring Custom-Built Skylights

Custom skylights don't have standard sizes, so precise manual measurement is essential. You'll need to measure width, height, and depth — and record the roof pitch angle.

Measure width at three points (top, middle, bottom) and height at three points (left, centre, right). Use the narrowest width and longest height.
1

Measure the Width

Measure the inside width of the skylight frame (not the glass — the frame opening):

  • Measure at the top, middle, and bottom
  • Record all three measurements in millimetres
  • The narrowest measurement is your order width

Custom skylights are often slightly trapezoidal (wider at the bottom than the top) due to roof framing. This is normal — the narrowest width ensures the blind fits through the frame at every point.

2

Measure the Height

Measure the inside height from the top rail to the bottom rail:

  • Measure at left, centre, and right
  • Record all three in millimetres
  • The longest measurement is your order height

We use the longest measurement so the blind fully covers the glass from edge to edge. Any excess is absorbed by the side channels.

3

Measure the Frame Depth

This is the distance from the glass surface to the inside front edge of the frame. Measure at all four corners — frames can warp over time.

⚠️ Shallow frame? If your frame depth is under 25mm, external mounting brackets may be an option — but they change the appearance. We recommend a free on-site assessment for shallow frames.
4

Check for Obstructions

Before you put the ladder away, check for anything that would interfere with a blind:

  • Handles or latches — do they protrude into the frame opening?
  • Winding mechanisms — chain winders or crank handles on manually operated skylights
  • Ventilation slots — some skylights have trickle vents at the top rail
  • Condensation channels — drainage channels at the bottom of the glass

Note the size and position of any obstructions. We may need to adjust the blind to clear them.

Roof Pitch & Angle

Unlike vertical windows, skylights are installed at an angle — and that angle matters for blind selection and performance.

Without side channels, blind fabric sags away from the glass due to gravity. Channels are essential for angled installations.

How to Measure the Angle

You have two options:

  1. Phone app: Most smartphones have a built-in inclinometer (on iPhone: open Measure → tap Level; on Android: search "Bubble Level" in your app store). Hold the phone flat against the skylight frame.
  2. Digital angle finder: More accurate. Place it against the frame and read the angle.

What the Angle Means

Roof Pitch Angle Blind Requirements Best Blind Type
Flat / very low 0°–15° Tensioned system essential. Standard blinds won't work — fabric pools on the glass. Honeycomb with tension wires
Low pitch 15°–25° Side channels + tension system recommended. Fabric still tends to bow. Honeycomb with side channels
Standard pitch 25°–45° Side channels sufficient. Most common Australian roof pitch range. Honeycomb or roller with channels
Steep pitch 45°–70° Side channels optional but recommended. Blind stays close to glass naturally. Any type works well
Near-vertical 70°–90° Treat as a standard vertical window. Channels optional. Standard blind — no special skylight system needed
💡 Melbourne standard: Most residential roofs in Melbourne sit between 22° and 30° pitch. If your skylight follows the roof line, expect to be in the "standard pitch" category — side channels required but no tension wires.

Which Blind Type for Your Skylight?

Not all blind types work well at an angle. Here's an honest breakdown based on what we install most:

Honeycomb / Cellular Blinds — Top Pick

The best all-rounder for Australian skylights. The cellular structure is lightweight (won't sag), provides genuine thermal insulation (R-value 2.0+ for double cell), and operates smoothly in side channels at any angle from 15° to 90°. Available in blockout or light-filtering fabrics.

  • Best for: bedrooms, living areas, any angle
  • Energy benefit: reduces heat gain through skylights by up to 45%
  • Frame depth needed: 25mm minimum

Roller Blinds — Budget-Friendly

Work at angles above 25° with proper side channels. The fabric rolls onto a tube at the top (or bottom for bottom-up operation). Heavier than honeycomb, so more prone to sagging at low angles. Good if you want a simple, clean aesthetic.

  • Best for: standard-to-steep pitch, cost-conscious installs
  • Frame depth needed: 35mm minimum
  • Limitation: not ideal below 25° pitch

Venetian Blinds — Light Control

Aluminium venetian blinds can work in skylights with side channels and tension cords. They give you adjustable light control — angle the slats rather than fully opening or closing. Heavier than honeycomb, so better for steeper pitches.

  • Best for: offices, kitchens, areas where adjustable light matters
  • Frame depth needed: 40mm minimum
  • Limitation: not suitable below 30° pitch; can rattle in wind

External Awning Blinds — Maximum Heat Block

Mounted on the outside of the skylight, these block solar heat before it hits the glass — far more effective than internal blinds for heat control. Professional installation only.

  • Best for: west-facing skylights, extreme heat areas, top-floor rooms
  • Reduces heat gain by up to 72% (vs 45% for internal honeycomb)
  • Requires: professional install, weather-rated hardware

Common Mistakes

⚠️ Measuring the glass, not the frame. The blind mounts inside the frame, not against the glass. Measure the frame opening — glass dimensions will be smaller and give you a blind that's too narrow.
⚠️ Forgetting the depth. A 25mm-deep frame won't fit a roller blind that needs 35mm. Always measure depth. It's the most commonly missed measurement for skylights.
⚠️ Ordering a standard blind for a skylight. Standard vertical blinds don't have side channels or tensioning. At any angle, the fabric will hang away from the glass and flap. You need a purpose-built skylight system.
⚠️ Ignoring the operation type. Centre-pivot skylights rotate the sash 180° for cleaning. The blind must clear this mechanism — some blind positions will block the pivot. Always note whether your skylight is centre-pivot, top-hung, or fixed.
⚠️ Rounding measurements. Skylight frames have tighter tolerances than standard windows. A 3mm error can mean the side channels don't align. Always measure in millimetres, never round.

Frequently Asked Questions

Do I need to measure if I have a VELUX skylight?

Not for width and height. Find the code plate on your skylight frame — it has a model code like MK04 or CK02. This code tells manufacturers the exact dimensions. You'll still need to note the frame depth and any obstructions.

Where is the code plate on a VELUX skylight?

Open the skylight and look at the top of the frame (the rail closest to the ridge). The code plate is a small metal plate with the model code, manufacture date, and serial number. On older models (pre-2013), it may be on the right-hand side of the frame.

What type of blind works best for skylights?

Honeycomb (cellular) blinds are the most popular for skylights in Australia. They're lightweight, provide excellent insulation (reducing heat gain through roof windows by up to 45%), and run smoothly in side channels at any angle. Blockout honeycomb blinds are ideal for bedrooms; light-filtering for living areas.

Can I install skylight blinds myself?

For VELUX-compatible blinds, yes — they clip into pre-existing mounting brackets. For custom skylights, professional installation is strongly recommended. Working overhead at angles is genuinely difficult, and incorrect mounting can damage the skylight seal or void your warranty.

What is the minimum depth needed for skylight blinds?

Honeycomb skylight blinds need 25–30mm frame depth. Roller skylight blinds need 35–40mm. If your frame is too shallow, external mounting brackets are available for some systems but change the appearance. See our honeycomb blinds measurement guide for more depth details.

Do skylight blinds need side channels?

Yes, for any skylight installed at an angle (which is most of them). Side channels hold the blind fabric against the glass so it doesn't sag or bow away from the window. Without channels, fabric will droop — especially with roller blinds. The only exception is near-vertical skylights above 70°.
