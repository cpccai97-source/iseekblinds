Measure Guide S-Fold / Wave Curtains

How to Measure for
S-Fold Curtains

S-fold curtains hang on a specialist track with evenly spaced gliders — which means measurement precision matters more than with other curtain styles. Get the track length and drop right, and you'll have those clean, rippling waves for years.

Time 10–15 min
Difficulty Easy
Tools Needed Tape, pencil, paper
Measurements needed Track width + Drop

What Makes S-Fold Different to Measure

S-fold (also called wave, ripple, or S-wave) curtains create their signature uniform folds because each glider on the track is spaced exactly the same distance apart — typically 70–100mm. That mechanical regularity means:

  • Fabric fullness is fixed — usually 2.0–2.5× track length (most suppliers default to 2.2×)
  • Track length is what you measure — the supplier calculates fabric width from that
  • You need a specialist S-fold track — standard curtain track or rods won't produce the wave effect
  • Stack-back is predictable — roughly 25–30% of track length per stack side

Unlike eyelet or pencil pleat curtains, you don't need to calculate fullness yourself. Just give us the track length and drop — we handle the rest.

Step-by-Step: How to Measure

  1. 1

    Decide mount type first

    Before picking up your tape measure, settle on ceiling mount or wall mount. For ceiling mount, confirm you have adequate clear ceiling above the window (S-fold tracks need about 80–100mm bracket clearance; double tracks need 120–140mm).

    💡 Pro tip

    If your windows are standard height (2100–2400mm ceiling), ceiling mount gives the most dramatic result with minimal extra cost — the track is simply longer.

  2. 2

    Measure the window opening width

    Using a steel tape measure, measure the window frame or opening width from inside edge to inside edge. Write this down — it's your starting point, not your track length.

    Add the following to get your finished track length:

    • Overlap: 150–200mm each side beyond the window frame (so curtains cover the architrave/wall when closed)
    • Stack-back: if you want curtains entirely off the glass when open, add another 200–300mm each side
  3. 3

    Measure the drop

    From your chosen track position, measure straight down to the floor. Use the adjustments below:

    FinishAdjustmentNotes
    Standard clearance−10 to −15mmCurtain clears floor when moving
    Sill-length−20mm below sillRarely used with S-fold — floor-length looks better
    Soft break+20 to +30mmGentle pooling at floor — dramatic, requires vacuuming
    Full puddle+80 to +120mmClassic couture look — high maintenance
    ⚠️ Don't measure in the middle

    Floors are rarely perfectly level. Measure drop at 3 points — left edge, centre, right edge of the window. Use the longest measurement as your drop if pooling; use the shortest if you want consistent clearance.

  4. 4

    Check ceiling recess depth (ceiling mount only)

    If mounting to the ceiling, ensure you have adequate clear depth above the window recess for the track bracket:

    • Single S-fold track: minimum 80–100mm depth
    • Double track (sheer + blockout): minimum 120–140mm depth
    • In-cornice hidden track: depends on cornice depth — send us photos if unsure
  5. 5

    Note the number of panels

    Most windows use a 2-panel split (left + right, meeting in the centre). For very wide windows or one-way draw, specify:

    • 2-panel split draw: standard, curtains meet in the middle
    • 1-panel one-way draw: all fabric stacks to one side (suits floor-to-ceiling glazing)
    • 3+ panels: for very wide openings (4m+), fabric may be split across multiple panels to reduce individual panel width
  6. 6

    Record and submit

    Write all measurements in millimetres. For each window, note:

    • Track width (mm)
    • Drop (mm)
    • Mount type (ceiling / wall)
    • Panel configuration (2-way split / 1-way draw)
    • Room / window label (e.g. "Living Room — North window")

    That's everything we need to quote. We calculate fabric width, number of panels, and glider count from your track measurement.

Common Measurement Mistakes

⚠️ Measuring fabric width instead of track length

The single most common error. We need your track length — the distance the track will span. We calculate the fabric width (track × fullness ratio) for you.

⚠️ Forgetting stack-back space

If your track ends at the window frame edge, the curtains will partially cover the glass when open. Extend the track 150–300mm past the window on each side so curtains stack cleanly on the wall.

⚠️ Using a fabric tape measure

Fabric tapes stretch. Always use a rigid steel tape measure. For long spans (3m+), get a helper to hold one end — the tape sags in the middle if you're measuring alone.

⚠️ Ordering S-fold curtains for a standard rod

S-fold curtains require a specialist track with evenly spaced gliders. Standard curtain rods, eyelet rods, and basic tracks won't create the wave effect. If you're replacing existing curtains, confirm you have (or are ordering) a compatible S-fold track.

  • No. S-fold curtains need a specialist track with evenly spaced gliders (every 70–100mm). The regularity of those gliders is what creates the uniform wave. A standard rod or basic track won't work — the curtains will bunch unevenly and lose the S-fold effect entirely.
  • For a clean float-above-floor look, deduct 10–15mm from the floor measurement. If you want the curtains to sit on the floor with a soft break, add 20–30mm. Avoid a gap between 15–40mm — it looks like a measuring error rather than an intentional design choice.
